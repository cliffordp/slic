# Xdebug in `tric`

## Available Commands

### `tric xdebug help`

List the available Xdebug commands.

### `tric xdebug status`

See if Xdebug is enabled or disabled, the host information, and the path mapping to add to your IDE.

### Enable/Disable Xdebug

1. `tric xdebug on`
1. `tric xdebug off`
1. Within `tric shell`:
    1. `tric xon`
    1. `tric xoff`
